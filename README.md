Application de gestion de billet pour des spectacles.

Lors du premier lancement de l'application, configurer un fichier .env et le relier à une base de données
MySQL.

Utiliser ensuite la commande php artisan migrate --seed pour mettre en place les rôles et permissions de base ainsi que les permiers utilisateurs qui permettent les tests.
